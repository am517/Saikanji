SaiKanji (彩漢字)

A colorful stroke-order font for kanji.

美しい色の筆順フォント — 漢字のために。

SaiKanji turns KanjiVG stroke data into a modern hybrid OpenType font with per-stroke color, stroke-order numbers, optional guide grids, and broad app/browser compatibility.

SaiKanjiは、KanjiVGのストロークデータを基に、現代的なハイブリッドOpenTypeフォントへと変換します。各ストロークに色を付け、筆順番号を内蔵、オプションのガイドグリッドを搭載し、さまざまなアプリやブラウザで幅広く動作します。

Install

Install once, then use it like a normal font.

一度インストールすれば、通常のフォントと同じように使えます。

Desktop

デスクトップ

  1. Download a .ttf file from the GitHub releases.
  2. Open the file and install it with your system font manager.
  3. Select the SaiKanji style in Pages, design tools, Anki, or other apps.
  1. GitHub releases から .ttf ファイルをダウンロード
  2. ファイルをクリックして、システムのフォント管理ツールでインストール
  3. Pages、デザインソフト、Anki、その他のアプリでSaiKanjiのスタイルを選択

Web

Web

  1. Download a .woff2 file from the GitHub releases.
  2. Open the file and add it to your project
  3. Use the WOFF2 files with @font-face. Host the files with your site when possible so font loading is cacheable and predictable.
  1. GitHub releases から .woff2 ファイルをダウンロード
  2. ファイルをプロジェクトに追加
  3. @font-face でWOFF2ファイルを使用。可能な限りサイトと一緒にホスティングすると、フォント読み込みがキャッシュ可能で安定します。

Mac (homebrew)

Mac (homebrew)

  1. Install the SaiKanji Tap
    brew tap am517/tap
    brew trust am517/tap
  2. Find and install a font
    brew search font-saikanji
    brew install font-saikanji-...
